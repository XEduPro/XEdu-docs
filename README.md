# XEdu-docs

本仓库为XEduPro的官方文档。

XEduPro源自上海人工智能实验室的开源人工智能工具包XEdu。XEdu于2024年8月因故中止更新后，引发广泛关注。2025年8月，由原项目组负责人谢作如和核心成员邱奕盛、王海涛等人在各自的工作单位支持下，重新组织力量进行维护。

因XEdu项目新增了对国产算力的支持，并且与魔乐社区深度合作，功能大大加强，故更名为XEduPro。
